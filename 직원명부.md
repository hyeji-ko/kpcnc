성명, 직급, 소속, 업무 정보만 매핑 (이메일 제외)
중복 등록 방지 기능
🚀 사용 방법:
"참석자 추가" 버튼 클릭 → 직원명부 모달이 열립니다
직원 정보 입력 → 성명, 직급, 소속, 업무, 이메일을 입력하고 "등록" 버튼 클릭
직원 목록에서 행 클릭 → 해당 직원이 자동으로 세미나 참석 명단에 추가됩니다
